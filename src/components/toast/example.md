## Toast

> import { Toast } from 'wue';


| 参数           | 类型          | 说明  | 可选值| 默认值|
| ------------- |:-------------:| -----:|-----:|-----:|
|visible| Boolean | 是否可见|||
|icon| String | 图标| ||
|type| String | 类型| ||
|message| String | 弹出消息| ||


```html
 <wue-toast :visible="true" message="弹出成功"></wue-toast>
```
