## Icon

```
import { Icon } from 'wue';
```

#Header 参数

| 参数           | 类型          | 说明  | 可选值| 默认值|
| ------------- |:-------------:| -----:|-----:|-----:|
|type |String| 图标类型| ‘success’, ‘info’, ‘warn’, ‘waiting’, ‘success-no-circle’, ‘circle’, ‘info-circle’, ‘download’, ‘cancel’, ‘search’|
|size|Number| 图标大小||false|


```
<wue-icon type="success" :large="false"></wue-icon>
```
