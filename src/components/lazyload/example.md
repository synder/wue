## Lazyload

```
import { Lazyload } from 'wue';
```

#Lazyload 参数

| 参数           | 类型          | 说明  | 可选值| 默认值|
| ------------- |:-------------:| -----:|-----:|-----:|
|src|String| URL|
|alt|String| Img的Alt提示|
|class|String| img的样式名称|


```
<wue-lazyload src="/img/logo.png"></wue-lazyload>
<wue-lazyload src="/img/logo.png"></wue-lazyload>
```
