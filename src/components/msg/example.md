## Loadmore

```
import { Loadmore } from 'wue';
```

#Loadmore 参数

| 参数           | 类型          | 说明  | 可选值| 默认值|
| ------------- |:-------------:| -----:|-----:|-----:|
|icon|String| 显示图标|success, warn, info, waiting|
|title|String| 显示标题|
|desc|String| 显示描述|


```
 <wue-msg title="line" icon="warn">你好</wue-msg>
```
